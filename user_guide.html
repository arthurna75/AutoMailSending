사용자 가이드 — 주요뉴스 다이제스트

초대받은 이메일로 로그인해서, 관심 키워드에 맞는 뉴스를 매일 정해진 시각에 이메일로 받아보는 서비스입니다. 이 문서는 로그인부터 설정 변경까지 사용법만 다룹니다.

접속 주소: https://web-psi-rouge-56.vercel.app

1. 로그인하기

  1. 위 주소로 접속 → 이메일 입력 → "인증 코드 받기" 클릭
  2. 입력한 이메일로 인증 코드가 담긴 메일이 옵니다 (제목: "Your sign-in link")
  3. 메일 안의 숫자 코드를 드래그로 복사해서, 화면의 코드 입력란에 붙여넣기
  4. 직접 손으로 옮겨 적으면 자리 수를 잘못 입력하기 쉬우니 복사/붙여넣기를 권장합니다.
  5. 메일에 링크는 없고 숫자 코드만 있습니다 — 그 코드를 화면에 입력하는 것으로 로그인이 완료됩니다.
  6. "로그인" 클릭하면 내 설정 화면(대시보드)으로 이동합니다.
  7. 메일이 안 오면: 스팸함 확인 → 그래도 없으면 서비스 관리자에게 문의(초대가 안 되어 있으면 로그인 코드 자체가 발급되지 않습니다).

2. 설정 화면 (대시보드) 항목 설명

항목 설명
키워드 받고 싶은 뉴스 검색어. 입력 후 Enter(또는 "추가")로 추가, ×로 삭제. 최소 1개 필요.
키워드당 최대 / 전체 최대 다이제스트 메일 하나에 키워드별로 최대 몇 건, 전체 합쳐서 최대 몇 건까지 담을지.
네이버 뉴스 검색 / 구글 뉴스 RSS 둘 중 하나만 선택(기본값: 네이버). 구글 뉴스 RSS는 한글 키워드에서 결과가 비어있거나 형식이 깨지는 경우가 잦아 기본은 네이버를 권장.
본문 크롤링으로 보강 기사 요약이 부실할 때 원문 페이지에서 본문을 더 가져와 보강할지 여부.
AI 요약 사용 켜두면 각 기사를 OpenAI가 1~2문장으로 요약해줌. 꺼두면 원본 스니펫 그대로 표시.
발송 시각 / 타임존 매일 몇 시(자신의 시간대 기준)에 메일을 받을지.
수신 이메일 비워두면 로그인한 계정 이메일로 감. 다른 주소로 받고 싶으면 여기에 입력.
제목 템플릿 메일 제목 형식. {date} 부분이 그날 날짜로 자동 치환됨.
다이제스트 활성화 꺼두면 로그인은 되지만 자동 발송은 멈춤(휴가 등으로 잠시 안 받고 싶을 때).

설정을 바꾼 뒤에는 반드시 "저장" 버튼을 눌러야 반영됩니다.

동음이의어 키워드에 의도 설명 추가하기 (선택 사항)

"재수"처럼 여러 뜻(대학 입시 재수 / "재수없다" / 검찰 재수사 등)을 가진 키워드는 등록한 키워드 태그를 클릭하면 의도 설명을 입력하는 칸이 펼쳐집니다. 예를 들어 "재수" 태그를 클릭해서 "대학 입시 재수 관련"이라고 입력해두면, 그 뒤로는 AI가 기사 제목을 보고 무관한 기사("재수없다", 검찰 재수사 등)를 걸러내고 발송합니다.

예시 화면 — "삼천리"처럼 회사명이 여러 개일 수 있는 키워드
키워드
키워드 입력 후 Enter
추가
동음이의어가 있는 키워드는 태그를 클릭해 의도를 설명해주세요 (예: '재수' → '대학 입시 재수 관련'). 설명을 입력하면 AI가 제목을 보고 무관한 기사를 걸러냅니다.
개인정보 × AI × 🎯 삼천리 ×
자전거 회사 아님. 에너지, 수입차판매, 외식 사업 회사
  • 설명을 넣지 않은 키워드는 예전과 동일하게 필터링 없이 전부 받습니다.
  • 의도 설명이 있는 태그에는 🎯 표시가 붙습니다.
  • 설명을 입력한 뒤에도 반드시 대시보드 "저장"을 눌러야 반영됩니다.
  • 의도 설명을 등록한 키워드가 있으면 다이제스트 발송(또는 테스트 메일)이 평소보다 4~5분 정도 더 걸립니다 — AI가 그 키워드로 수집된 기사 제목을 하나하나 검토하는 과정이 추가되기 때문입니다. 즉시 오지 않아도 정상이니 여유를 두고 기다려주세요.

3. 지금 바로 테스트 메일 받아보기

"지금 테스트 메일 받기" 버튼을 누르면 매일 정해진 발송 시각까지 기다리지 않고, 지금 화면에 입력된 내용 그대로(저장 여부와 무관) 다이제스트 메일을 1통 미리 받아볼 수 있습니다.

  • 저장을 먼저 하지 않아도 됩니다 — 키워드나 옵션을 방금 바꾼 상태에서 바로 눌러도, 그 수정된 내용 기준으로 메일이 옵니다. 다만 화면을 새로고침하거나 나갔다가 다시 들어오면 저장하지 않은 변경 내용은 사라지니, 마음에 들면 꼭 "저장"도 눌러두세요.
  • 클릭 후 1~2분 정도 걸립니다(즉시 도착하지 않아도 정상입니다). 도착하면 스팸함도 한 번 확인해보세요.
  • 단, 의도 설명(🎯)을 등록한 키워드가 하나라도 있으면 AI 의도분석 처리 때문에 4~5분 정도로 더 오래 걸릴 수 있습니다 — 정상이니 조금 더 기다려주세요.
  • 이 테스트 메일은 그날의 정규 발송 여부와 무관합니다 — 테스트로 받았다고 해서 그날 예정된 자동 발송이 취소되지 않고, 그대로 정해진 시각에 또 받게 됩니다(같은 기사가 다시 나올 수도 있습니다 — 정상입니다).
  • 설정을 바꿀 때마다 결과를 바로 확인하고 싶으면 이 버튼을 여러 번 눌러도 괜찮습니다.

4. 지난 발송 이력 보기

대시보드 하단의 "발송 이력 보기" 링크(/history)에서 그동안 정규 발송된 다이제스트의 날짜/제목/ 기사 건수를 볼 수 있습니다. (테스트 메일은 이력에 남지 않습니다 — 3번 참고.)

5. 로그아웃

대시보드 오른쪽 위 "로그아웃" 버튼을 누르면 됩니다. 다시 쓰려면 1번부터 다시 로그인하면 됩니다 (설정은 그대로 남아있습니다).
